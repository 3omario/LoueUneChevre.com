# Equipe historique

Avant la levee de fonds, tout le projet etait porte par deux personnes.

## Nina Martin - developpeuse full-stack

Responsabilites de fait :

- site React ;
- API de devis ;
- prototype mobile ;
- demandes urgentes du support ;
- corrections de production quand Marc n'etait pas disponible.

Habitudes observees :

- merge direct sur `main` quand la demande vient du fondateur ;
- tests ajoutes seulement apres incident ;
- documentation ecrite apres coup.

## Marc Pelletier - administrateur systeme

Responsabilites de fait :

- VPS production ;
- nginx ;
- systemd ;
- backups ;
- deploiements manuels ;
- certificats TLS.

Habitudes observees :

- deploiement depuis son poste ;
- checklist papier ;
- corrections directes sur le serveur ;
- logs expliques dans Slack plutot que dans un runbook.

## Prochaine etape

La nouvelle equipe doit transformer ces pratiques individuelles en process collectif :

- PR obligatoires ;
- CI visible ;
- staging fiable ;
- runbooks versionnes ;
- metriques DORA ;
- post-mortems blameless.
